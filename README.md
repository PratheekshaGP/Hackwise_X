 Hackwise_X

 Overview:

This project builds an image classification model to distinguish between **Chihuahua dogs** and **Muffins**, a challenging visual classification problem.
The model is trained using **ResNet-18 with random initialization** following the competition rules.

 Dataset:
The dataset contains images labeled as:
* Chihuahua
* Muffin
* Undefined
  
Training:
Run the training script:
python /train.py

Prediction:
Generate predictions using:
python predict.py

Output file:
submission.csv

Technologies:
* Python
* PyTorch
* 3LC Data-Centric AI tools
* GitHub
